Client optimization, 22 may, 2025

    Optimized Client Performance
    • Performance Optimization - font loading
      • Added preconnect links to fonts.googleapis.com and fonts.gstatic.com to reduce connection setup time (~160ms saved).
      • Updated Google Fonts link to use modern css2 API with display=swap to prevent render-blocking and layout shifts.
      • Resulted in faster text rendering and improved Lighthouse scores (FCP, LCP, CLS).
